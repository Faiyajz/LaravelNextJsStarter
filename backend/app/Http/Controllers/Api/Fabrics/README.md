# Fabrics Module

**Purpose**: Fabric CRUD, listing, stocks, barcodes, and trash/restore.

## Responsibilities
- Fabric list/filter/pagination.
- CRUD operations and barcode creation.
- Stock adjustments.
- Trash/restore lifecycle.

- DTOs: `app/DTOs/Fabrics/`
- Policies: `app/Policies/Fabrics/`
