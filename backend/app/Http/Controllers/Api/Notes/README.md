# Notes Module

**Purpose**: Notes creation for supplier and fabric entities.

## Responsibilities
- Create notes for supported noteable types.
- Enforce permission checks before creation.

## Key Paths
- Controllers: `app/Http/Controllers/Api/Notes/`
- Requests: `app/Http/Requests/Notes/`
- Actions: `app/Actions/Notes/`
- DTOs: `app/DTOs/Notes/`
- Policies: `app/Policies/Notes/`
